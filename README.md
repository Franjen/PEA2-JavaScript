# PEA 2 - Sistema de Cotización de Computadoras

## Descripción

Este proyecto fue desarrollado para la asignatura Lenguaje Interpretado.

El sitio web permite realizar una cotización de computadoras mediante la selección de componentes, utilizando HTML5, CSS, JavaScript y jQuery.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery

## Funcionalidades

- Validación de formularios.
- Cálculo automático de cotización.
- Aplicación de IVA.
- Aplicación de descuentos.
- Manipulación del DOM.
- Eventos con jQuery.
- Diseño adaptable.
